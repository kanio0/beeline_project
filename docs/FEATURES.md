# Реализованные фичи

## Роли
- volunteer
- organizer
- superadmin

## Авторизация
- email/password + JWT
- Telegram-бот для профиля и быстрых команд
- архитектурно готово для Telegram Login Widget

## Волонтёры
- список
- фильтрация
- поиск
- статус активности
- личный кабинет
- история активности

## Команды
- рейтинг команд
- лидер команды
- суммарные коины
- активность команды

## События
- список и календарный вид
- участие
- подтверждение участия
- фотоотчёты
- экспорт в Google Calendar через `.ics`

## Рейтинг и коины
- расчёт по формуле из ТЗ
- leaderboard пользователей
- leaderboard команд
- учёт anti-spam, streak, quality, scale

## Достижения
- новичок
- активист
- организатор
- ручные и авто-выдачи
